New, Open and Save

Tap on New to create a new document.

Tap on Open to open an existing document.

The document you are working on is auto-saved so there is no Save button.

WriteRT works with Markdown documents (.md).

Focus Mode

Start typing to enter focus mode. Press ESC, move the mouse or tap the screen to exit focus mode.

When in focus mode, WriteRT will, by default, highlight the current sentence you are writing. This can be disabled by accessing Settings from the Settings charm or changed to highlight the current paragraph or line.

Spell checking is disabled when in focus mode to avoid interrupting your flow. As soon as you exit focus mode, the spell checker will highlight all errors for review.

Markdown

Markdown is an extremely easy way to format text without relying on menus. The text is simply decorated with the appropriate markup: *this is how you make text italic* for example.

Your documents are saved in plain text format and exported as HTML.

On the app bar, tap on Markdown Cheat Sheet for an easily accessible Markdown reference.

Preview

Tap on Preview for a quick preview of your document as HTML. Tap anywhere on the screen outside of the preview popup to dismiss it.

Word Goal

If you want to write a particular number of words, you can setup WriteRT to notify you when you reach that number: access Settings from the Settings charm, toggle Word Goal on and select your goal.

Searching

Use the Search charm to find text in the document. Tap on the Search button again to find the next match.

Exporting and Sharing

The Copy button on the app bar copies the document to the clipboard. The Export HTML button on the app bar exports a copy of the document rendered as HTML.

Besides the ability to export your document as HTML or copy it to the clipboard, you can also share it using the Share charm with any apps that accept HTML sharing (for example you can easily email a document by sharing it with the Mail app).

Tips

  • Syntax highlighting in the editor is minimal - only titles, bold and italics are highlighted. Use the preview button to see how the text looks like as HTML (with lists, blockquotes, code etc.)
  • Save your documents on SkyDrive so you can access them from all your devices.
  • Your highlighting preferences and word goal are automatically synchronized across devices.
