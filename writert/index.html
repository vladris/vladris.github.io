Simple. Inspiring. Beautiful.

WriteRT is a focused writing application with Markdown for formatting. As soon as you start typing, WriteRT gets out of your way - the UI fades out and you can focus on writing, with no distractions.

  • No Save button because all changes are auto-saved as you write.
  • No complex menus for formatting text - formatting is done with Markdown syntax.
  • Share data with other apps through the Sharing charm.
  • Set a word goal and get a notification when you reach it.
